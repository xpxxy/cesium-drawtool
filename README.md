### A Simple Drawing Utility Tool For Cesium

using callback style


### Usage 

```bash
# clone
git clone https://github.com/xpxxy/cesium-drawtool.git  

# install
pnpm install

# might be needed
pnpm approve-builds

# run
pnpm run dev
```
